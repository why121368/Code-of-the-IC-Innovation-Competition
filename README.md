### 项目简介
本作品的目标是基于雨珠S套件开发一套用于测试兼容BQ769x0系列芯片的电池管理芯片智能交互测试系统，该系统能够对待测芯片进行有效的功能测试和多种参数读取测试。测试系统能够实现待测芯片需要完成的测试，包括对待测芯片的监测、保护和控制三类关键功能的测试。在测试过程中可以通过键盘鼠标或语音控制两种方式，实现对芯片功能模块的测试并配置功能参数，实现不同工作条件下的芯片测试。作品系统实现数据上云，可以在无人情况下有效监测芯片的工作状态并且及时报警。

本项目代码在此开源，供大家交流讨论，代码中包括Python & Verilog ：
UI设计代码
界面中控件的代码（主界面代码,控件代码）
测试过程中的代码（IO控制代码，Uart&IIC通信代码，控制代码）
以及一些调试代码
